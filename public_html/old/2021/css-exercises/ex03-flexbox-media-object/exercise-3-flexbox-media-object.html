Exercise 3

This is Exercise #3, lecture on CSS Flexbox, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is to practice configuring flexbox, including item properties. There is a popular pattern in web development, called "media pattern". The idea is to display an image and text side by side, without wrapping the text around the image. An example could be a comment on a blog article. In this exercise you will create a flexible container for displaying the media object and a text.

Instructions

Create an HTML document with an image and a text or reasonable length (how about Emmet expression lipsum800 ?). You can use images from Lorem Picsum. Align them as in the media object pattern:

  1. Image on the left, text on the right.
  2. Image should fill 30% of the available width, the text: 70%.
  3. Minimum width for the image: 150px. Minimum width for the text column: 350px.
  4. Both image and text containers should dynamically expand if there is more space available.
  5. Image should have correct proportions.
  6. The image should have 20px margin on the sides (not on the top).
  7. If there is not enough horizontal space available, a scrollbar should appear. I.e., the text should not wrap under the image.
  8. Now try to align the image on the right. If you have done it correctly, it should be a single additional line of CSS code.

The following CSS properties may be handy (for the container):


        display: flex;
        flex-wrap
        flex-basis
        flex-grow
        flex-shrink
        align-self
        order
        min-width
        margin
