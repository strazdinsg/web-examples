Exercise 11

This is Exercise #11, lecture on using the jQuery library, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is practice using the fancy features of jQuery library.

Instructions

Part 1 - replacing a set of elements with jQuery

  1. Create a new HTML document. Include jQuery library in it. There are several ways to do that. Suggested way: include this external .js file: https://cdnjs.cloudflare.com/ajax/libs/jquery/3.5.1/jquery.min.js
  2. Create five text paragraphs with any text inside. Hint: if your IDE supports emmet, you can write the following expression to get 5 paragraphs with Lorem Ipsum text with `0 words in each paragraph: (p>lipsum10)*5.
  3. When the document is loaded, run a jQuery code which replaces text in all paragraphs on the page with "jQuery woz ere". Hint: in jQuery you can use $(document).ready(function() {...}) to execute code when the document is loaded. See jQuery documentation.
  4. Add a button to the page with text "Throw dice".

Part 2 - send Ajax request with jQuery

Re-do the exercise # 9 (throwing a dice) using jQuery. Instead of using the XMLHttpRequest, use $.get() to send an HTTP GET request to the server, then display the dice value, as you did in exercise # 9.
