Exercise 4

This is Exercise #4, lecture on traversing DOM with Javascript, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is learn how you can access specific elements in the DOM tree.

Instructions

Part 1 - inspect an existing site in the console

Go to https://web-tek.ninja in your browser (preferably in another tab), open developer tool console and explore the DOM:

  1. type document.body in the console, hit Enter. What do you see?
  2. Select the heading "CSS exercises" and save the reference to it in a variable heading1. You can do that by issuing the following command in the console: var heading1 = document.getElementById('css-exercise-heading'); Now change the text of the heading to "Cool exercises".
  3. Find the ID of the heading "CSS Grid exercises". Hint: you can right-click on it with the mouse, select Inspect. The Elements tab of dev tools will be opened, and the element selected. Do you see the id="..."?
  4. Now change the text of this heading (from the previous step) to "Hei!".

Create your first DYNAMIC websites!

  1. Create a page which has one paragraph (<p>) element in it. The text inside it should be "Loading...". Now add a Javascript to the page, which sets the text in the <p> element to "Hello" when the page is loaded.
  2. Add a paragraph to the same page (or create a new page if you want) that shows a text: "X seconds have passed since the page was loaded‚Äù. Update X every second.
  3. Create a "calculator":
    1. Two text inputs (let us call them A and B), the user can input a number in each of them. Set A=3, B=5 by default.
    2. A text saying: "the two numbers multiplied are X".
    3. Update the X to be A*B whenever text changes in any of the inputs.
    4. Extra challenge: show "Not a number!" when either A or B (or both) are not numbers.
